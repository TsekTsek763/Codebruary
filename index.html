Codebruary 2025 - Full Timeline

Day 1: Print 'Hello, World!'

✅ Use print or console.log to display text.
print('Hello, World!')

Day 2: Ask for user's name and greet them.

✅ Use input() and f-strings.
name = input('Enter your name: ')
print(f'Hello, {name}!')

Day 3: Add two numbers entered by the user.

✅ Convert input to int, then add.
a = int(input('Enter first number: '))
b = int(input('Enter second number: '))
print('Sum:', a + b)

Day 4: Check if a number is even or odd.

✅ Use modulo operator %.
num = int(input('Enter a number: '))
if num % 2 == 0:
    print('Even')
else:
    print('Odd')

Day 5: Find the largest of 3 numbers.

✅ Use if-elif-else or max().
a = int(input())
b = int(input())
c = int(input())
print('Largest:', max(a, b, c))

Day 6: Calculate factorial using loop.

✅ Use a for loop.
n = int(input())
fact = 1
for i in range(1, n+1):
    fact *= i
print('Factorial:', fact)

Day 7: Reverse a string.

✅ Use slicing [::-1].
text = input('Enter text: ')
print('Reversed:', text[::-1])

Day 8: Check if string is palindrome.

✅ Compare with reversed.
text = input('Enter text: ')
if text == text[::-1]:
    print('Palindrome')
else:
    print('Not palindrome')

Day 9: Generate multiplication table.

✅ Use a loop up to 10.
num = int(input('Number: '))
for i in range(1, 11):
    print(f'{num} x {i} = {num*i}')

Day 10: Count vowels in a string.

✅ Loop and count vowels.
text = input('Enter text: ')
vowels = 'aeiouAEIOU'
count = sum(1 for ch in text if ch in vowels)
print('Vowels:', count)

Day 11: Print all prime numbers from 1 to 50.

✅ Use a prime checking function.
for num in range(2, 51):
    for i in range(2, int(num**0.5)+1):
        if num % i == 0:
            break
    else:
        print(num, end=' ')

Day 12: Find the length of a list.

✅ Use len().
mylist = [1, 2, 3, 4]
print('Length:', len(mylist))

Day 13: Sort a list.

✅ Use sort() or sorted().
nums = [3, 1, 4, 2]
nums.sort()
print('Sorted:', nums)

Day 14: Check if an element is in list.

✅ Use 'in' keyword.
item = 3
mylist = [1, 2, 3, 4]
print('Found' if item in mylist else 'Not found')

Day 15: Sum all elements in a list.

✅ Use sum().
nums = [1, 2, 3]
print('Total:', sum(nums))

Day 16: Create a dictionary and print keys.

✅ Use keys().
mydict = {'a': 1, 'b': 2}
print('Keys:', list(mydict.keys()))

Day 17: Access a dictionary value.

✅ Use dict[key].
mydict = {'name': 'Alice'}
print(mydict['name'])

Day 18: Loop through a dictionary.

✅ Use items().
mydict = {'a': 1, 'b': 2}
for k, v in mydict.items():
    print(k, v)

Day 19: Define a function to square a number.

✅ Use def and return.
def square(x):
    return x * x
print(square(5))

Day 20: Use try-except for division.

✅ Catch ZeroDivisionError.
try:
    a = int(input())
    b = int(input())
    print(a / b)
except ZeroDivisionError:
    print('Cannot divide by zero')

Day 21: Read from a file.

✅ Use open() and read().
with open('file.txt', 'r') as f:
    print(f.read())

Day 22: Write to a file.

✅ Use write() method.
with open('file.txt', 'w') as f:
    f.write('Hello file')

Day 23: Create a class with attributes.

✅ Use class and __init__().
class Person:
    def __init__(self, name):
        self.name = name
p = Person('Bob')
print(p.name)

Day 24: Create a class with method.

✅ Use def inside class.
class Person:
    def greet(self):
        print('Hi!')
p = Person()
p.greet()

Day 25: Use a for loop with range.

✅ Use range().
for i in range(5):
    print(i)

Day 26: Use a while loop.

✅ Increment conditionally.
i = 0
while i < 5:
    print(i)
    i += 1

Day 27: Use a list comprehension.

✅ Simple syntax.
squares = [x**2 for x in range(5)]
print(squares)

Day 28: Use zip() to combine lists.

✅ Use zip().
a = [1, 2]
b = ['one', 'two']
print(list(zip(a, b)))
